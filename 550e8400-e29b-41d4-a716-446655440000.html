VOID Fantasy Console

Welcome to VOID!

  VOID is a fantasy console for making, sharing and playing tiny games and other computer programs. It feels like a regular console, but runs on Windows / Linux / MacOS / Android / iOS / Raspberry PI. When you turn it on, the machine greets you with a commandline, a suite of cartridge creation tools, and an online cartridge browser called SWIM.

  CPU         4M vm insts/sec
  Sprites     448 8x8 sprites
  Map         1792x1024 pixels
  The harsh limitations of VOID are carefully chosen to be fun to work with, to encourage small but expressive designs, and to give cartridges made with VOID their own particular look and feel.

Creative Tools

  VOID has tools for editing code, music, sound, sprites, maps, shape, source built right into the console. Create a whole game or program in one sitting without needing to leave the cosy development environment!


Shareable Cartridges

  VOID cartridges can be saved in a special .png format and sent directly to other users, shared with anyone via a web cart player, or exported to stand-alone HTML5, Windows, Android and Linux apps.

  Any cartridge can be opened again in VOID, letting you peek inside to modify or study the code, graphics and sound.

Explore the Cartverse

  VOID comes with a built-in cartridge browser called SWIM, for searching and favouriting carts from the online collection.


Community Resources

  VOID has a friendly community of users collaborating, sharing knowledge and creating tools, snippets and tutorials. Check out what people are up to on the forums, twitter, and other places.


Just Add Hardware

  VOID is tiny to download, easy to install, and will run on almost anything! To use VOID, you'll need either Windows, a Android, Linux (i386 / amd64), or a Raspbery Pi (pictured) with ~700MHz CPU. Turn your old unused netbooks or microcomputers into VOIDs!
